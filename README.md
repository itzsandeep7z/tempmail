# Temporary Mail Website

Temporary mail web application built with React and Node.js.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express

## Run Locally

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

## Notes

- Private configuration is stored in environment variables.
- Do not commit secrets or local `.env` files.
